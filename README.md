RecruitAI is an AI-powered Resume Parsing and Candidate Management System that automatically extracts candidate information from PDF resumes using Large Language Models (LLMs) and stores the structured data in a PostgreSQL database.

## Features

* 📄 Upload PDF resumes
* 🔍 Extract text from resumes using PyMuPDF
* 🤖 Analyze resumes using LLMs (Cerebras GPT-OSS-120B, Ollama, Gemini)
* 📊 Extract structured candidate information
* 🛠 Extract candidate skills
* 📂 Extract project details and descriptions
* 🗄 Store candidate data in PostgreSQL
* 🚫 Prevent duplicate candidate entries
* 🎨 User-friendly Streamlit interface
